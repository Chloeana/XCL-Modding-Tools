X-Change™ Life's Unofficial Modding Tools

Welcome to X-Change™ Life's Unofficial Modding Tools

These tools are intended to serve as a coding assistant, to facilitate working with the code files.

Choose a tab above to get started.

Some general tips for using these tools:

  • Always double-check the tooltips for detailed guidance and examples.
  • In each tab, start from the top and move downwards. Lower sections can sometimes be dependant on higher section inputs.
  • When building a Playable Character, ensure you read each section carefully to help with expectations.
  • Media filenames must correlate with the specified sections in each tab. Double check your paths and prefixes!!
  • For sections with multiple media files, ensure your filenames match what is outlined in that section. Filenames are important!!


If you have questions or suggestions, put them in the XCL Discord!

This is only intended to make the coding portion easier. Please refer to the walkthrough for specific instructions.
